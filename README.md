
---

## 环境要求

* **Unity 版本**: `2021.3.31f1`
* **依赖包**:
    * `com.unity.xr.picoxr` (PICO Unity Integration SDK) - **注意**：本项目通过本地路径引用此 SDK，你可能需要自行下载并导入。

---

## 如何运行

1.  **克隆仓库**:
    ```bash
    git clone [你的仓库地址]
    ```
2.  **切换分支**:
    本项目主要开发分支为 **`dev`**。
    ```bash
    git checkout dev
    ```
3.  **打开 Unity 项目**:
    使用 Unity `2021.3.31f1` 版本打开项目。
4.  **运行主场景**:
    在 Unity 编辑器中打开以下场景，然后点击运行：
    `Assets/Release/Scenes/HandSandPC_Dev.unity`
